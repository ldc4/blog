LeetCode No.127 Word Ladder

2016-03-14 Kommentare Arithmetic

翻译：
给出两个单词（start和end）和一个字典，找到从start到end的最短转换序列
比如：
1.每次只能改变一个字母。
2.变换过程中的中间单词必须在字典中出现。

start = “hit”
end = “cog”
dict = [“hot”,”dot”,”dog”,”lot”,”log”]
一个最短的变换序列是”hit”->”hot” -> “dot”-> “dog”->”cog”
返回它的长度 5

注意:

  • 如果没有转换序列则返回0。
  • 所有单词具有相同的长度。
  • 所有单词都只包含小写字母。
